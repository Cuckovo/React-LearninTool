项目简介

伴学知识库是一套以 AI Agent 为核心、学科工具为辅助的交互式学习系统。最初从 MathLearningTool 重构而来，目标是用 AI 帮你解题 + GeoGebra 帮你画图，打造一站式高数学习助手。

📐 GeoGebra — 动态数学引擎

全球领先的开源数学可视化工具 — 将抽象公式转化为交互式图形，让数学看得见、摸得着。

什么是 GeoGebra？

GeoGebra 是一个免费的动态数学软件，集几何、代数、微积分、统计于一体。输入一个函数表达式，立刻看到它的图像。拖拽、缩放、变换 — 所有操作都是实时的。

核心特性

📐 函数绘图：支持 f(x)、参数方程、极坐标、隐函数
📏 几何构造：点、线、圆、多边形 — 欧几里得几何全支持
📊 3D 可视化：立体几何、空间曲面直观展示
🧮 代数运算：CAS 符号计算引擎，支持求导积分
📱 全平台：Web / iOS / Android / Windows / Mac / Linux

core-feature · 重点介绍

🔗 跨页面联动 — 核心亮点

三大页面之间通过全局状态 Context + useReducer 实现数据互通，AI 对话的输出可以自动驱动 GeoGebra 绘图。

AI 对话 → GeoGebra 绘图联动全链路

👤 用户输入
"求 sin(x) 导数"
→
🤖 DeepSeek
返回结构化解析
→
🔍 AI Parser
提取 f(x)=sin(x)
→
📤 dispatch
SEND_PLOT_COMMAND
→
📐 GeoGebra
evalCommand()

RN ↔ GeoGebra 通信

Android: webViewRef.injectJavaScript(
  'ggbApplet.evalCommand("f(x)=sin(x)")'
)
Web: iframe.contentWindow.postMessage(
  {type:'evalCommand', expression:'f(x)=sin(x)'}
)

知识库 → AI 对话联动

大纲页点击「去学习」
→ dispatch CREATE_KNOWLEDGE_SESSION
→ 自动切换到 AI 对话页
→ 注入节点上下文 (label + standardDef)
→ 双模式：解题 / 知识库学习
